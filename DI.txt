# DI ( Dependency Injection : 의존 주입 )
- 객체간의 결합을 느슨하게 하는 스프링의 기술입니다
- 객체 사이의  의존관계를 자기 자신이 아닌 외부에 의해서 설정하는 방식입니다

# DI 방식 ( 의존 관계 관리 )
- 생성자 방식( Construction Injection )
  > 생성자를 통해서 의존관계를 연결시키는 것입니다
- 설정 메서드 방식 ( setter Injection )
  > '<property> 태그의 name 속성을 통해서 어떤 의존 객체가 주입되는 지 알 수 있습니다'
